# employee-task-management-system-java
Java application for managing employee tasks in a software company using OOP principles and a Swing-based graphical interface.
The system allows a project manager to create tasks, assign them to employees, monitor their status, 
and estimate the total work duration of each employee.

The application follows object-oriented programming principles and uses a layered architecture. 
It also supports both simple and complex tasks, where complex tasks can be composed of multiple 
simple or complex subtasks.
## Features
- Add and manage employees
- Create simple tasks with a start and end hour
- Create complex tasks composed of multiple subtasks
- Assign tasks to employees
- Modify task status (Completed / Uncompleted)
- Estimate the work duration of employees based on completed tasks
- View employee task statistics
- Filter employees with work duration greater than 40 hours
- Display statistics for completed and uncompleted tasks
## Technologies Used
- Java
- Java Swing
- Object-Oriented Programming
- Composite Design Pattern
- Java Serialization
